Comparación de Código HTML

Aquí puedes visualizar la versión original y la versión mejorada del código HTML con accesibilidad.

Código Antiguo

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Galería de Noticias ATOM</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha1/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <nav class="navbar navbar-dark bg-dark">
        <a class="navbar-brand" href="#">Noticias ATOM</a>
    </nav>
    <div class="container mt-5">
        <input type="text" placeholder="URL del feed">
        <button>Cargar</button>
        <div id="newsContainer"></div>
    </div>
</body>
</html>

Código Mejorado

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Galería de Noticias ATOM</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0-alpha1/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <header>
        <nav class="navbar navbar-dark bg-dark" role="navigation" aria-label="Menú principal">
            <a class="navbar-brand" href="#">Noticias ATOM</a>
        </nav>
    </header>
    <main class="container mt-5">
        <section aria-labelledby="feedInputLabel">
            <label id="feedInputLabel" for="feedUrl" class="visually-hidden">Ingrese la URL del feed</label>
            <input type="url" id="feedUrl" placeholder="URL del feed" aria-required="true">
            <button aria-label="Cargar noticias del feed">Cargar</button>
        </section>
        <section id="newsContainer" aria-live="polite"></section>
    </main>
</body>
</html>
